## ❓ Frequently Asked Questions

### Do we need to use BMAD for everything?
**No.** Use the scale-adaptive approach:
- **Quick Flow**: Bug fixes, small features (<5 min planning)
- **BMad Method**: Standard features (<15 min planning)
- **Enterprise**: Complex systems (<30 min planning)

### Will this slow us down?
**Short-term**: 2-3 hours more upfront planning per feature  
**Long-term**: 4-6 hours saved on rework and pivots  
**Net benefit**: 2-3 hours saved per feature after learning curve

### What if AI gives bad recommendations?
**You're in charge.** BMAD agents advise; humans decide. Always review and validate agent outputs.

### How long to become proficient?
- **Week 1**: Learning and friction
- **Week 2-4**: Building competence  
- **Month 2+**: BMAD becomes natural

### Do we need special tools?
Works with any IDE supporting AI assistants:
- ✅ Cursor
- ✅ Claude Code
- ✅ Windsurf
- ✅ VS Code (with Copilot)
