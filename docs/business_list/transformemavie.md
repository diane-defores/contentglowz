# 📊 Fiche Recap Business : Transforme Ma Vie

## Vue d'ensemble
**Transforme Ma Vie** est un service de coaching spécialisé dans la transformation personnelle, le bien-être et le bonheur. Créé par Diane Defores, il cible particulièrement les victimes de violences et les personnes souffrant d'addiction, en proposant une approche holistique combinant santé physique, équilibre énergétique (yin/yang) et psychologie des relations. Le site utilise une approche scientifique et pratique pour accompagner les individus vers le changement positif.

## 👥 Profil Client

### Persona Principal : "Le Reconstruisant Résilient"
- **Démographie** : 25-55 ans, mixte genre, toutes origines sociales
- **Psychographie** : En phase transition de vie, motivé changement positif
- **Pain Points** :
  - Souffrance violences (physiques/psychologiques)
  - Dépendances (substances, comportements)
  - Déséquilibre santé physique/mentale
  - Difficultés relations sociales
  - Manque sens/épanouissement personnel
- **Objectifs** : Reconstruire vie saine, retrouver autonomie, créer relations harmonieuses
- **Comportements** : Recherche aide professionnelle, suit communautés bien-être, teste approches holistiques

### Segments Cibles
1. **Victimes de Violences**
   - Profil : Femmes/hommes ayant subi violences conjugales/familiales
   - Besoins : Reconstruction confiance, sécurité émotionnelle
   - Sources : Associations, bouche-à-oreille, urgences médicales

2. **Personnes Addictes**
   - Profil : Dépendances substances (alcool, drogues) ou comportements (jeux, sexe)
   - Besoins : Sevrage accompagné, prévention rechute, reconstruction vie
   - Sources : Centres traitement, médecins, familles

3. **Entrepreneurs Burnout**
   - Profil : Business owners épuisés, stress chronique
   - Besoins : Équilibre vie pro/perso, gestion stress, productivité saine
   - Sources : Réseaux professionnels, recommandations

4. **Personnes en Transition**
   - Profil : Divorce, deuil, changement carrière, quête sens
   - Besoins : Accompagnement changement, développement personnel
   - Sources : Thérapies, livres développement personnel
