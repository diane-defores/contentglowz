## Problèmes de parcours
- **Trous** : Pas de page dédiée MOFU (comparatif, case study, demo). Pas de checkout. Pas d'onboarding.
- **Culs-de-sac** : Pricing buttons sans lien. Liens sociaux footer → `#`. Articles blog sans CTA produit.
- **Sauts** : Le blog (TOFU) n'a aucun pont vers le MOFU/BOFU — le visiteur organique lit et part.
- **Orphelines** : Aucune après l'audit SEO (toutes les collections sont routées).

## Flux de conversion principal (actuel)
```
[Google/Social] → Blog article → (dead end, pas de CTA)
[Direct/Referral] → Homepage Hero → Problem → How It Works → Features → Pricing → (dead end, button cassé)
```

## Flux de conversion souhaité
```
[Google/Social] → Blog article → CTA banner "Try ContentFlow" → Homepage ou Signup
[Direct/Referral] → Homepage → Problem → Who It's For → How It Works → Pricing → Checkout
[Indécis] → Newsletter signup ou Free resource → Email nurture → Conversion
```
