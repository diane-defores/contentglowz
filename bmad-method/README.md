## 🎯 What is BMAD Method?

The BMAD Method is an AI-driven agile development framework that provides:

- **19+ Specialized AI Agents** - Experts in PM, Architecture, Development, Testing, etc.
- **50+ Guided Workflows** - Structured processes for every development phase
- **Scale-Adaptive Planning** - Automatically adjusts to project complexity
- **Complete Lifecycle** - Analysis → Planning → Architecture → Implementation

## 🚀 Quick Start

### Installation
```bash
npx bmad-method@alpha install
```

### First Steps
1. **Initialize Project**: Load Analyst agent → Run `*workflow-init`
2. **Choose Track**: Select Quick Flow, BMad Method, or Enterprise Method
3. **Follow Workflows**: Agent guides you through planning and implementation

### Common Commands
- `*workflow-init` - Initialize project and select track
- `*workflow-status` - Check current progress
- `*prd` - Create Product Requirements Document
- `*architecture` - Design system architecture
- `*sprint-planning` - Plan next sprint
- `*dev-story` - Implement a user story

## 📊 Our Implementation

### Project Track
We use the **BMad Method Track** for this project because:
- Complex multimodal chat application
- Multiple integrated features (artifacts, streaming, auth)
- Requires comprehensive architecture planning
- 20-50+ user stories expected

### Active Agents
- **PM (Product Manager)** - Requirements and feature planning
- **Architect** - System architecture and technical design
- **Developer (DEV)** - Code implementation
- **Test Architect (TEA)** - Testing strategy and automation
- **Scrum Master (SM)** - Sprint planning and story management
- **Tech Writer** - Documentation

### Workflow Status
Track project progress in: `docs/bmad-method/bmm-workflow-status.yaml`
