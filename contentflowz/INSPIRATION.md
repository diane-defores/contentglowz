# Inspiration — ContentFlowz

## Vidéo IA

- **Remotion** — Génération vidéo programmatique en React, open source, composable, rendu serveur
- **Runway ML** — Génération et édition vidéo par IA, Gen-2 text-to-video, référence du marché
- **Pika Labs** — Text-to-video, interface simple, résultats impressionnants
- **Synthesia** — Avatars vidéo IA, présentations automatisées, cas d'usage entreprise
- **HeyGen** — Avatars vidéo, clonage de voix, traduction automatique
- **Luma AI** — Capture 3D, génération vidéo IA, Dream Machine

## Audio IA

- **Eleven Labs** — Voice synthesis de pointe, clonage vocal, Sound Effects, musique IA
- **Suno** — Génération musicale par IA, text-to-music, qualité impressionnante
- **Descript** — Édition audio/vidéo par le texte (transcription puis édition du texte = édition du média)

## Image IA

- **Midjourney** — Qualité artistique de référence, communauté Discord
- **DALL-E** (OpenAI) — Intégration API, édition d'images, inpainting
- **Stable Diffusion** — Open source, personnalisable, LoRA, ControlNet
- **Adobe Firefly** — Suite créative IA intégrée à Photoshop/Illustrator

## Design et édition

- **Canva** — Référence UX pour le design simplifié, templates, collaboration
- **CapCut** — Édition vidéo mobile, templates viraux, effets tendance

## Points clés à retenir

| Source | Leçon |
|---|---|
| Remotion | Le code comme outil de création vidéo — composable et versionnable |
| Canva | La simplicité d'usage prime sur la puissance des fonctionnalités |
| Descript | Éditer du média via du texte est un paradigme puissant |
| Eleven Labs | La qualité vocale IA a atteint un niveau quasi indistinguable de l'humain |
| Runway ML | Le text-to-video va transformer la création de contenu |
| Stable Diffusion | L'open source permet une personnalisation impossible ailleurs |
| CapCut | Les templates sont le meilleur onboarding pour les non-experts |
