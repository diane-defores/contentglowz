# Branding — ContentFlowz

## Identité

- **Nom** : ContentFlowz (Content + Flow + z pour l'écosystème)
- **Tagline** : "Crée du contenu, laisse l'IA faire le reste"

## Identité visuelle

### Cohérence écosystème

- S'inscrit dans l'écosystème Flowz (rainbow spectrum de WinFlowz)
- Accent créatif propre : dégradés vibrants, effets de génération (pixels, ondes, particules)

### Mode d'affichage

- **Dark mode par défaut** — environnement de création professionnelle
- Light mode disponible mais secondaire

### Ambiance

- Futuriste mais accessible
- Créatif et expérimental
- Effets visuels qui évoquent la "génération" (particules, morphing, pixels qui se forment)

## Palette de couleurs

| Couleur | Utilisation |
|---|---|
| Noir profond (`#0A0A0F`) | Fond principal (dark mode) |
| Violet électrique (`#7C3AED`) | Accent principal, actions IA |
| Cyan néon (`#06B6D4`) | Accent secondaire, audio |
| Magenta (`#EC4899`) | Accent tertiaire, image |
| Vert émeraude (`#10B981`) | Succès, export terminé |
| Dégradés vibrants | Éléments hero, headers, CTA |

## Typographie

- **Titres** : Sans-serif bold, moderne (Space Grotesk ou similaire)
- **Corps** : Sans-serif lisible (Inter)
- **Interfaces de génération** : Monospace (JetBrains Mono ou Fira Code) pour les prompts et paramètres

## Icônes

Chaque type de contenu a son icône dédiée :

| Type | Icône |
|---|---|
| Vidéo | Film / caméra |
| Audio | Onde sonore / micro |
| Image | Cadre / appareil photo |
| Texte | Document / plume |
| Animation | Mouvement / frames |
| IA | Étincelle / cerveau |

## Animations

- **Génération en cours** : shimmer effect, progressive reveal (l'image/vidéo apparaît progressivement)
- **Particules** : effet de particules lors du lancement d'une génération
- **Morphing** : transitions fluides entre états (prompt vers résultat)
- **Loading states** : skeleton screens avec shimmer, jamais de spinner simple
- Toutes les animations respectent `prefers-reduced-motion`

## Sous-identités par outil

Chaque outil peut avoir sa propre couleur d'accent, mais doit rester dans le cadre ContentFlowz :

| Outil | Couleur d'accent |
|---|---|
| Vidéo (Remotion) | Violet (`#7C3AED`) |
| Audio (Eleven Labs) | Cyan (`#06B6D4`) |
| Image | Magenta (`#EC4899`) |
| Animation | Orange (`#F97316`) |

## Valeurs de marque

| Valeur | Ce que cela signifie |
|---|---|
| Créativité | L'outil amplifie la créativité humaine, il ne la remplace pas |
| Innovation | Toujours à la pointe des modèles et techniques IA |
| Accessibilité | Pas besoin d'être expert pour créer du contenu de qualité |
| Expérimentation | Essayer, itérer, apprendre — chaque prototype est une exploration |
