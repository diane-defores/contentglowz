# Sources — ContentFlowz

## APIs IA

### Voix et audio

- **Eleven Labs API** — Synthèse vocale, clonage de voix, musique IA, podcast automatisé
- **OpenAI Whisper** — Transcription audio-to-text, support multilingue

### Texte et langage

- **OpenAI GPT API** — Génération de scripts, prompts, sous-titres, descriptions
- **Anthropic Claude API** — Rédaction, analyse, structuration de contenu

### Image

- **OpenAI DALL-E API** — Génération d'images, édition, variations
- **Stability AI API** — Stable Diffusion, inpainting, img2img, ControlNet
- **Replicate** — Accès à des centaines de modèles open source (Flux, SDXL, etc.)

### Vidéo et animation

- **Remotion** — Framework React pour la génération vidéo programmatique
- **FFmpeg** — Traitement vidéo/audio en ligne de commande (encodage, conversion, montage)
