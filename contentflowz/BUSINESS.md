# Business — ContentFlowz

## Mission

Explorer et prototyper les outils de création de contenu IA de nouvelle génération.

## Proposition de valeur

Suite d'outils expérimentaux pour créer du contenu multimédia (vidéo, audio, image, animation) assisté par IA — sans expertise technique requise.

## Statut

**Phase exploratoire** — prototypes multiples, pas encore de produit unifié.

### Prototypes actuels

| Prototype | Technologie | Type de contenu |
|---|---|---|
| Remotion template | React + Remotion | Vidéo programmatique |
| Eleven Labs generators | Eleven Labs API | Audio, musique, podcast |
| AI image generation | Stable Diffusion / DALL-E | Images |
| AI animation studio | Divers modèles | Animation |
| Design concepts | Astro / React | Maquettes et UX |

## Business Model (futur)

### SaaS freemium

- **Gratuit** : accès limité aux outils (watermark, résolution réduite, X générations/mois)
- **Pro** (~15 EUR/mois) : limites étendues, export HD, pas de watermark
- **Business** (~49 EUR/mois) : usage commercial, API access, priorité de génération

### Bundle créateur

- Accès à toute la suite ContentFlowz à prix réduit vs outils individuels
- Intégrations entre outils (générer une image puis l'animer, ajouter une voix off)

### API

- Exposer les pipelines de génération pour d'autres développeurs
- Tarification à l'usage (par génération, par minute de vidéo/audio)

## Persona principal

**Le Créateur Pressé**
- Créateurs de contenu, marketeurs, solopreneurs
- Veut produire du contenu multimédia de qualité
- N'a pas d'expertise en montage vidéo, design ou production audio
- Cherche des outils rapides et intuitifs

## Marché

- **AI content creation** : 10 milliards USD+ d'ici 2028
- **Segments** : vidéo IA (croissance la plus forte), audio IA, image IA
- **Tendance** : démocratisation de la création — passage de l'outil pro au self-service
